## 📖 Descripción del Proyecto
**MicroC** es un pre-compilador desarrollado como proyecto para el curso de Autómatas y Lenguajes. El objetivo es simular el comportamiento básico de un compilador para el lenguaje C, permitiendo escribir código, gestionar archivos y verificar errores sintácticos simples.

### El sistema analiza y valida:
* ✅ Existencia de la función principal `int main()`.
* ✅ Correcto uso y balanceo de llaves `{ }`.
* ✅ Presencia obligatoria de punto y coma `;`.
* ✅ Validación básica de estructura.

Incluye además control de edición, con confirmación de salida con cambios sin guardar y una ventana de ayuda.

## ▶ Instrucciones de Ejecución

### Opción 1: Ejecutar desde el proyecto
Abre una terminal dentro de la carpeta del proyecto y ejecuta:
dotnet run

### Opción 2: Ejecutar versión compilada
Navega a la carpeta de publicación:
cd bin/Release/net8.0/linux-x64/publish/
./MicroC
